Additional Tips:
  • Working with Adobe Illustrator: This project involves some copy & pasting items from one AI file to another. When you paste call out items into your new map group them before you paste them into the template so things can remain in order and be easily edited or deleted.
  • Lock Layers that are not being used.
  • Reposition labels so they are not covered by call out items.
  • Use the following colors and line-weights:
    • Class 1 (Bike Path) Color: Leaf Green (RGB 56-168-0 or #38a800), Line Weight: 2
    • Class 2 (Bike Lane) Color: Mars Red (RGB 255-0-0 or #ff0000), Line Weight: 2
    • Class 2.1 (Bike Lane one side only) Color: Mars Red (RGB 255-0-0 or #ff0000), Line Weight: 1
    • Class 3 (Bike Route) Color: Electron Gold (RGB 255-170-0 or #ffaa00), Line Weight: 2
    • Class 3A (Arterial Bike Route) Color: Lapis Lazuli (RGB 0-92-230 or #005ce6), Line Weight: 2
    • Class 3B (Bicycle Boulevard) Color: Dark Amethyst (RGB 132-0-168 or #b400a8), Line Weight: 2
    • Class 4 (Protected Bike Lane) Color: Macaw Green, Line Weight: 2
    • <All Other Values>, Two cartographic line symbols, Line Weight: 1.5
