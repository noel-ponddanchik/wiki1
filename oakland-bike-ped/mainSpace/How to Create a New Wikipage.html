Create a New Page

  1. To make a new page click on the plus sign next to 'Pages and Files' in the left hand navigation bar.
  2. Then in the 'Make a New Page' window fill in
    • Page Name > Create a descriptive and intuitive title
    • Template > Select 'Wikipage' from the drop down to use the official wiki template
    • Tags > Add a few sensible tags to each new page to improve searching capabilities, e.g. 'how to' or 'signage'
      • Tags are able to include more than one word, to create a tag type in the tag and hit enter
    • Once you are satisfied with your selections select the 'Create' button


Edit Template to Populate New Page

  1. Use the descriptions and tips within the template to guide where you place information in your new wikipage
  2. Replace the directions within the template with your own content
  3. To link to another page or website use the 'Link' button in the edit tool bar
  4. Explore the widgets available by selecting the 'Widget' button in the edit tool bar
  5. Insert images or files by selecting the 'File' button in the edit tool bar
  6. Tables can be created by selecting the 'Table' button in the edit tool bar

Other Tips

  • Make sure to save your changes
  • Include links to relevant pages
