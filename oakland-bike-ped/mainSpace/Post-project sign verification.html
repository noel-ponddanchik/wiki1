Post-project sign verification instructions


1. Gather sign plan list and map (photocopy docs in folder), clipboard, measuring tape/wheel, orange vest, writing implements, Jenniferâ€™s business card, phone.

2. Review plan, map out route.

3. Look at each sign and document any problems being sure to review for all of the following:

a. wrong location

b. too low/too high

c. gap between blades

d. wrong message or incorrect arrows/logos

e. too much or not enough space between logos and text and/or edge of sign

f. signs not removed

g. bad location choice by us

4. Photograph and get GPS coordinate for each sign using mobile App. Photos should show the full length of the pole to the top of the sign.

5. Back in the office, update the following fields in Access: installed checkbox, Last Verified date, Photo (ID.jpg) and, if applicable, installation notes (for mistakes are too trivial to fix).

6. If needed, create a punch list for items needing repair/fixing. The punch list template is the last tab in \\oakland\pwa\Transportation\Bike-Ped_Program\Projects\Bikeways-signage\1-resources\work-orders-general\work-order-overview-template.xls, a multi-tab file that is created for all corridor projects, and filed by corridor.

7. File the photos by ID# in \oakland\pwa\Transportation\Bike-Ped_Program\Projects\Bikeways-signage\SignPhotos (e.g. 547.jpg). If lighting was poor, use Photoshop to lighten before filing.

8. Link to GIS. See http://oakland-bike-ped.wikispaces.com/Update+Rack+and+Signage+GIS+Layers+After+Fieldwork.
