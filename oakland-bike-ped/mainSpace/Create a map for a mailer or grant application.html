Look at previous mailers (\\oakland\pwa\Transportation\Bike-Ped_Program\Outreach\Mailers).

Use an old print I [bike] oakland bikeways map and pencil in a box around the proposed map extent. Discuss the extent and orientation with project manager before proceeding. Also, discuss what bikeway types the particular map will show. Maps will typically include two or more of the following bikeway categories (and it will vary depending on the "story" we're trying to tell):
  • Project
  • Existing
  • Pending construction
  • Under development
  • Proposed

Open this file in ArcView:
"\\oakland\pwa\Transportation\Bike-Ped_Program\Outreach\Mailers\Mailer-map-template.mxd"
  • Zoom to the agreed upon extents
  • Right click to adjust rotation, symbolize bikeway types, etc. Bikeway types are symbolized generally (existing, project, under development, proposed, planned) as needed to tell the story.
  • Template has line types formatted in the likely version for print; plus, they are in shades of CMYK black so they can be easily selected in Illustrator to change globally if needed.
  • All the bikeway line types other than "Existing" will need to be changed in Illustrator, some globally, some "by hand"
  • When final, Bookmark extent (in case you need to redo)
  • Export Map to .ai format, save to desktop temporarily

In Illustrator, open .ai format map
  • As prompted, Update
  • First thing: Select, Object, Text Object: then, on keyboard, arrow left, then arrow right. (This fixes an issue with font sizes.
  • Save in mailer folder (or grant folder, if for grant application), and retitle; file type can be .ai, .pdf, .eps (don't recommend .png or .jpg which are harder to edit)
  • Open the Layers palette and keep it open; helpful to lock layers and select just what you need and nothing else
  • Use Select, Same, Stroke/Color/Weight to adjust line types, weights, colors by group
  • Lock layers that won't change (like curb face)
  • Copy a legend from another mailer project and tweak to select the categories and line types desired.
  • Using the Direct Select tool, select the section of “project bikeway(s)” being highlighted. Change the line type (stroke, color, dashed/solid) to differentiate it.
  • Using the Direct Select tool, click on any segments of proposed bikeways that will not be included on this particular map and delete them. Click on remaining proposed bikeways and change line types as needed to reflect the appropriate categories
  • Delete duplicate or un-needed street and park names
  • Add or delete labels, street names, etc.
  • Save when complete
