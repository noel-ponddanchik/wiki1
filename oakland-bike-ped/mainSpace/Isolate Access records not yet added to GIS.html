In syncing our access database to GIS layers it is essential both datasets match. This process allows us to identify points (typically for installed racks and signs) not yet added in GIS.

To check the data do as follows:

-copy current access table to new excel file
-export current GIS attribute table to .txt file
-add GIS .txt table to new sheet in the excel file using "add text" button at top left of data tab
-sort GIS layer by ID (rack id, sign id...)
-highlight and copy all GIS rows in ID field
-Paste GIS ID field to bottom of sheet containing current access data under the same ID field
-The run conditional formatting


Use the conditional formatting feature to highlight all duplicate rows, then filter by cell color from the right click menu, then delete rows from the right click menu:

1. Select all items
2. Conditional Formatting -> Highlight cell rules -> Duplicate Values -> OK
3. Right click in selection -> Sort -> Sort by selected cell's color
4. Right click in selection -> Delete Row -> OK

-once conditional formatting is run sort by color to isolate the non duplicates that then should be added or deleted from GIS
