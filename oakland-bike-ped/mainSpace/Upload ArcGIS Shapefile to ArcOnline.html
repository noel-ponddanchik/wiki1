Export Shapefile from ArcMap to a Folder

  • Create a file folder in a convenient location and name it the shapefile you intend to export
  • In Arcmap, right click the layer you want to export in the table of contents
  • Go to Data and click Export Data
  • Select the new folder you created for the Output feature class:, choose shapefile for save as type:
  • Press Save and the OK
  • Do not add this exported layer to your map

Zip Shapefile

  • Navigate to your folder with the exported shapefile in Windows Explorer
  • Right click the folder go to 7-zip and select Add to "filename.zip"

Upload to ArcOnline

  • Sign into ArcOnline and navigate to My Content
  • Click the Add Item menu and select From my computer
  • Press the Choose File button and select your zipped shapefile
  • Add a few tags and then publish
  • Edit metadata and share!
