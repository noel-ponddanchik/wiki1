Development of the annual "I [bike] Oakland" bikeways map is, generally,a three-month process though notes are maintained throughout the year as ideas and observations are made by staff. Following are detailed instructions:

Year round
  • Maintain a running list of updates in the directory for the map year in \\oakland\pwa\Transportation\Bike-Ped_Program\Outreach\Maps\IbikeOakland
  • Track the number of boxes mailed to individual shops and given to the Mayor’s office, distributed at events, etc. using a spreadsheet in the map year folder named 20XX-production-distribution. This is, in part, to determine if the map quantity needs to be increased for the subsequent year.
  • Track the totals from year to year in this spreadsheet: \\oakland\pwa\Transportation\Bike-Ped_Program\Outreach\Maps\IbikeOakland\Ibike-map_production_ongoing.xls
  • Monitor quantity and evaluate the need to reprint. Do not reprint after March 1st.

NOTE ON CALCULATING MILEAGE:
As of 2015, the map designer, Jake Coolidge, is calculating the total map mileage, including that for adjoining jurisdictions. The bikeways database reliably calculates the mileage for Oakland (run database report rBikeway_network_mileage_summary and use existing centerline miles). Mileage for the bikeways in adjoining jurisdictions has proved difficult, in part because it needs to include only the mileage shown on the map. Jake is under contract through 2016. Should there be a different designer in the future who cannot calculate the mileage for adjoining jurisdictions, following are past instructions for doing so, should they prove helpful:

GIS Layer (adj_bikeways): \GIS\master.gdb
Map location: \GIS\adjoining_jurisdictions\printMap_totals.mxd

1. Update ‘adj_bikeways’ to reflect any changes in the past year
2. Create a new clipped layer from updated ‘adj_bikeways’:
  • Open ArcToolbox > Analysis Tools > Extract > Clip
  • Input Features = adj_bikeways
  • Clip Features = printMap_boundary
  • Output Feature Class = adj_bikeways_clip(n)

3. Calculate mileage from the new layer:
  • Open adj_bikeways_clip(n) attribute table
  • Select by attribute where “EXCLASS” <> 0
  • Right-click on field Shape_Length > Statistics
  • Take the sum (ft) divided by 5280[b1] (the number of feet in a mile)
