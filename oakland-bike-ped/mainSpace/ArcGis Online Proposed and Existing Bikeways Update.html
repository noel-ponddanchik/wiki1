The Bike Program's public-facing web map shares the status of existing bicycle facilities (bikeways, signs, counts, and racks). The map also features pop-up windows that include photos of existing bike parking and bike signs. First launched in September 2016, the map should be updated every six months in January and July. This wiki requires an understanding of the relationship between the various relevant Access databases as well as ArcGIS and some intermediate-level Excel functions.Files for this project are stored at Bike-Ped_Program\GIS\ArcGISOnline-webmap.

The process for updating the map is as follows:

1. Create New Project Folder in the ArcGIS-Online Webmap Folder
The folder can be found here:

\\oakland\pwa\Transportation\Bike-Ped_Program\GIS\ArcGISOnline-webmap

The created folder should say the month and year of the update, example: May_2017

2. Open Access
This step is done for each of the Layers that are updated (Purposed Bikeways and Existing Bikeways)
First check to see that the access database is up to date. Meaning that the newest and most recent iteration of the database is being used. Secondly export the Database as an excel file, and save as a .csv. This excel file will eventually be used to join with its corresponding Shapefile.

3. Open ArcMap
Open ArcMap and add the shapefile that you are looking to update (Bikeways). These shapefiles can be found in our Master Geodatabase.

4. Join Excel File
Take the Excel file that you created in Step 2 and join the Excel file to the Shapefile used in the previous step. The joined file should have all the information from the shapefile plus all the info from the Excel file.

5. Export the Shapefile
Right Click the shapefile you are planning on updating and export the shapefile to your working folder.

6. Zip the Shapefile
The Shapefile you are working with must be zipped for it to be displayed by the ArcOnline application. To zip a shapefile, select all the elements (except the LOCK file) that were created by exporting from ArcMap and then right click on all the selected files and select the “send to” option. Within the “send to” option click Compressed Zipped Folder. The Shapefile is now ready to be uploaded.

7. Add Zipped Shapefile to ArcGIS webmap to replace existing layers
Go to https://www.arcgis.com/home/index.html and log on using Jennifer's log on.
Click the "Content" tab at the top.
Launch the ArcGIS online map that you will be editing (City of Oakland Bikeways: Existing and Proposed Bikeways) (use the Web Map version not the web mapping application).
  • Click to open the map page.
  • Click "Open in Map Viewer"
Then add the new Zipped Shapefile by doing the following,
  • Click the "Add" tab in the top left
  • Click "add layer from file"
  • Browse for your desired Zipped Shapefile Folder and select it

You will need to add the bikeways zipped shapefile twice, once to symbolize Proposed and once to symbolize Existing class.
Copy the symbology from an existing layer by going to the content tab then hovering over the layer you just added and clicking the symbol that looks like shapes.
Copy the design of the pop-up windows to match the old layer.
After updating the layers and symbology, you can delete the old layer and rename the new layer with the proper name.
MAKE SURE TO save the map before closing the webpage (on top toolbar).
