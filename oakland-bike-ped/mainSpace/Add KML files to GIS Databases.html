Use this how to when one needs to get kml files off of a smartphone and into a geodatabase.

Table of Contents






  1. Take the individual kml files and merge them into one kml file.
    1. Create a copy of one of the kml files and open the kml file in a text editor such as Notepad++.
    2. Open each of the other kml files and copy all of the information within, and including, the placemark tags.
    3. One at a time, paste the copied placemark tags and information from step 1b in between the closing placemark tags in the file from step 1a and before the closing document tag (</Document>).
    4. Save this kml file with a new name such as “All_points_for _Genoa_West_St.kml”
  2. In ArcMap, add the feature class to the map document that you would like to add the kml points to.
  3. In the ArcToolbox, go to conversion tools, then click “From KML”, then click KML to Layer.
  4. Next to Input KML File, click the folder button and navigate to the kml file created in step 1d.
  5. Choose a location for the output layer, and give the output layer a name. Add the output layer to the map.
  6. Right click the layer we want to add the kml points to and go to edit features àstart editing.
  7. Right click the output layer created in step 5, press control + c, then press control + v. When the dialog box asks where you want to paste the new features, click the layer that we selected in step 6.
  8. Make sure the attributes of the newly created features are correct and then save edits.
