1. Create Buffer Around Transit-Related Wayfinding Destinations

  1. To create a buffer in ArcGIS, press ctrl+f to open arc's search function.
  2. Type buffer in the search bar and click the Buffer (Analysis) tool
  3. For Input Features select the points representing your transit stops or entrances, for Output Feature Class name the file something sensical (ie bart_buffer) and then key in the desired Distance of the buffer and press OK

2. Calculate Number of Bike Parking Spaces

  1. Use the bike parking shp file caled parking located here: \\oakland\pwa\Transportation\Bike-Ped_Program\GIS\master.gdb
  2. Join this layer to the tBikeParking table located in the BikeParking.accdb database. Information for connecting to this database
  3. Select by Attributes : Status = 'Installed'
  4. Right click on the parking layer in the table of contents, Selection > Create Layer from selection to make a layer with just the installed bike racks
  5. Using the buffers you have created Select by Location
  6. Selection method: 'select features from'; Target layer: pick your layer with the installed bike racks; Source layer: pick your transit buffer; press OK
  7. Right click on the installed bike racks layer in the table of contents, click Open Attribute Table, then press Show selected records
  8. In the table scroll to the BikeCapacity column and right click, select Statistics and look at the Sum to see the total amount of bike parking spaces that fall within the buffer
