This page is a style guide for the preliminary project map. It includes information about fonts and colors to be used in the map.

Font:

'Gill Sans MT' 8pt

Color:

  • All Outlines = RGB 0-0-0 (black)
  • ‘Project’
    • Green Background Fill = RGB 13-108-83 (or CMYK: 95 M: 31 Y: 77 K: 20)
    • White Font = RGB 255-255-255
  • ‘Project Regulatory’
    • White Background Fill = RGB 255-255-255
    • Black Font = RGB 0-0-0
  • ‘Project Warning’
    • Yellow Background Fill = RGB 255-255-0
    • Black Font = RGB 0-0-0
  • ‘Other’ and ‘Other –Regulatory’
    • Grey Background Fill= RGB:178-178-178
    • White Font = RGB 255-255-255
