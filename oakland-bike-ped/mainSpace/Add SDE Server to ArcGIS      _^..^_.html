Acquire SDE Server Mirror

SDEuser.sde
1. Download SDEuser.sde file above and save to your workstation Desktop.

Connect to SDE Server

1. Open ArcCatalog (separate from ArcMap).
2. Connect to folder > Desktop
3. Expand the Desktop connection, and then right click the SDEuser.sde image > copy
4. Scroll down the Catalog menu, find "Database Connections" and then right click > paste
5. The SDE Server will appear (you may need to right click > refresh "Database Connections")
6. Double click the SDE Server icon (with red 'x') to access.
