This project reflects the increase in bikeway mileage and the changes to bikeway types (Class) starting in 1976 (when the first bike route signs are thought to have been installed). The maps are posted on the Completed Projects page of the bike programâ€™s website (www2.oaklandnet.com/government/OAK024591 and used in various outreach efforts as desired.
Specific outputs are:
  1. Individual .pdf maps for each calendar year starting in 1976, except for those years between 1977 and 1996 when there were no changes to the network: 1977, 1979, 1981-86, 1989, 1991, 1992, and 1994-96.
  2. Packet of the above maps, updated annully
  3. Individual .png maps for web animation at http://www2.oaklandnet.com/Government/o/PWA/s/BicycleandPedestrianProgram/Map1/index.htm.
  4. Tables and charts illustrating the changes based on data in the timelapse database.

The project requires that updates be made in four locations: (1) timelapse database; (2) timelapse GIS layer/geodatabase; (3) bikeways segments database; (4) bikeways GIS layer (master geodatabase).

Updates will be made in January (or February) of each year.
