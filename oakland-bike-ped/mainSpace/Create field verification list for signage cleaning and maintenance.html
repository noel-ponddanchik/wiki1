The bike program uses volunteers to inspect previously installed signs based along wayfinding signage corridors. To facilitate the field work, a template linked to the sign database is used to generate an inspection form by corridor.


Create and print template

  1. In WayfindingSigns access database, select desired corridor and any other signs to be included. Make sure all columns are visible.
  2. Copy and paste desired corridor data into an Excel sheet. Update the sort order to reflect the likely field work order. For now, make sure the column headings in signsToVerify-columns are in the spreadsheet (some are no longer in the database). Data is only necessary for the columns through Sign Message 3. Note that the Photos column is now the last column.
  3. Move the Photos column data to the last spreadsheet column and enter \\oakland\pwa\Transportation\Bike-Ped_Program\Projects\Bikeways-signage\SignPhotos\ in every column cell.
  4. Create new column ‘@IMAGE to the right of ID column.
  5. In the new column, enter: =CONCATENATE(“AD2,A2,".jpg") and copy down to the rest of the column. The resulting cell content will be \\oakland\pwa\Transportation\Bike-Ped_Program\Projects\Bikeways-signage\SignPhotos\xxx.jpg
  6. Create a project specific folder in \\oakland\pwa\Transportation\Bike-Ped_Program\Projects\Bikeways-signage\3-signs-to-fix-or-document\verification (based on the corridor name, e.g. "Genoa-West")
  7. Save the Excel sheet as a CSV file in the new folder.
  8. Open InDesign file: \\oakland\pwa\Transportation\Bike-Ped_Program\Projects\Bikeways-signage\3-signs-to-fix-or-document\verification\verification-template.indd
  9. Along the top menu, go to Window – Utilities – Data Merge and a new panel should pop up. Click the further options button and click “Select Data Source”. Find and select the CSV file for your corridor. The column headings from the CSV files should show up in a list
  10. Go to the data merge panel, click the drop down and select “Create merged document”.
  11. Under “Records per Document Page” select “Multiple Records”.
  12. Check “Preview Multiple Record Layout” to make sure stuff shows up.
    (Above instructions excerpted from http://gcostudios.com/2012/designing-and-data-merging-with-multiple-records-in-indesign-cs6/)
  13. If images are too large, resize each manually by selecting the ‘select’ tool (black arrow) and resizing the image box while holding SHIFT (retains scale) so that each image fits on the page.
  14. Save file into project-specific folder
  15. Export as PDF and save..
  16. Double-check that photos and sign messages are matched.
  17. Print PDF.


Send the volunteer to verify signage.


Back in the office, enter information in database, in particular:
  • Last Verified (include for all signs inspected)
  • Repair_Status
  • Repair_Instructions.

If a sign is missing and will not be replaced, make sure to uncheck the “Installed” checkbox, change the corridor to Z-Removed, and document observations in the Notes field.
