1. Split the segment(s) in the bikeways access database to get the new ID number for the segment(s).
  2. Open bikeways.mxd and enable editing.
  3. Right click on the bikeways layer, and choose Selection, Make this the only selectable layer.
  4. Zoom in and select the segment to be split. Using the Split tool, click on the split point. Change the ID attribute of the new resulting segment in the bikeways layer. Record the length of the new and original, now shortened, segment.
  5. Save edits.
  6. In the bikeways access database: (a) enter the new/changed segment lengths; and (b) record the changes in the TimelapseToAdd field.

To add a new bikeway segment that wasn't in the proposed bikeway network:
  1. Add new segment to the bikeways access database to get the new ID number.
  2. Open bikeways.mxd.
  3. Select the new segment(s) from the OaklandStreets layer. Hit Control C (copy).
  4. Highlight the bikeways layer in the table of contents. Hit Control V (paste). (You might have to first enable editing.) A dialog box will appear. Confirm that the bikeways layer is the one being edited.
  5. Edit the ID attribute of the new segment to match the new one in the database..
  6. Save edits.
  7. In the bikeways access database: (a) enter the new/changed segment lengths; and (b) record the changes in the TimelapseToAdd field.
