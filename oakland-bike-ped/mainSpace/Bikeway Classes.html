Bikeway classes are a categorization scheme to define the various types of bikeways. Each class has specific design characteristics that are fundamental to the planning, feasibility, design, and user experience of each facility type.

Overview

The City of Oakland's Bicycle Master Plan (2007) proposes a bikeway network of approximately 245 miles in length that is composed of five bikeway types:
  • Bike Path (Class 1)
  • Bike Lane (Class 2)
  • Bike Route (Class 3)
  • Arterial Bike Route (Class 3A)
  • Bicycle Boulevard (Class 3B)

The Class 1, 2, and 3 bikeways are defined by the Caltrans Highway Design Manual, Chapter 1000. Oakland's Bicycle Master Plan adds Class 3A and 3B to this scheme. Three additional classes - Classes 0, 2.1, and 4 - are used for record keeping purposes (and they are defined below).

The bikeway database tracks both existing and proposed bikeways, each in a separate field. Two fields are needed to capture all possible combinations, as suggested by these examples:
  • A street with no bikeway (Existing Class 0) is proposed to get a bike lane (Proposed Class 2).
  • A street with an existing bike route (Existing Class 3) is proposed to be upgraded to a bike boulevard (Proposed Class 3B).
  • A street with an existing bike route (Existing Class 3) is proposed to be removed from the bike network (Proposed Class 0).

Definitions of Existing & Proposed Classes


Value
 Name
 Description
0
 No bikeway
 Either no bikeway exists or no bikeway is proposed for a particular roadway segment. For Existing Class 0, the Proposed Class is generally not 0. In some instances an existing bikeway (Existing Class does not equal 0) is proposed to be removed (Proposed Class 0). Segments with both Existing Class 0 and Proposed Class 0 were analyzed as part of the 2007 Bicycle Master Plan but not included as part of the bike network.
1
 bike path
 Provides a paved right-of-way that is completely separated from the street. Bike paths are typically shared with pedestrians and are often called mixes-use paths.
2
 bike lane
 Striped on streets and designated specifically for bicyclists. They are Oakland's preferred treatment for busy streets.
2.1
 Bike lane on one side only
 This designation is used for one bike lane on only one side of a two-way street. Note that a one-way street with one bike lane is designated as Class 2. Class 2.1 is used to imply unfinished business - a bike lane is still needed on the street.
3
 bike route
 Have shared travel lanes and are designated because they provide better connectivity than other nearby streets. Bike routes are designated with bikeway guide signs and may have no other bicycle-specific features. The 2007 Bicycle Master Plan proposes bike routes only in the Oakland Hills above Mountain Blvd due to the recreational nature and challenging topography of this area. The bikeways in the rest of the city are proposed for bikeways that provide a higher level of accommodation (Class 1, 2, 3B).
3A
 arterial bike route
 Located on busier streets where bike lanes are not possible and parallel streets do not provide adequate connectivity. They include sharrows and bicycle-specific signage. Arterial bike routes are used to make near-term connections in the bike network. Over time they should be replaced with better facilities (e.g., Class 2).
3B
 bicycle boulevard
 Prioritizes bicyclists on quiet residential streets with traffic calming, sharrows, and bicycle guide signs.
4
 Proposed bikeway
 This class is for cycletracks aka protected bikeways. (Note: Class 4 was once a way to identify "proposed bikeways with wide travel lanes or low traffic volumes that provide useful connections in their current form" on the annual print map.)
